About Me

I'm a full-stack developer, experienced in React, Redux, Vue.js, Ruby on Rails, and plain-ol-JavaScript.

Coming from research and academia, what drew me to being a software engineer was the algorithms and problem solving. However, these days, I get just as excited about building something that is interactive, easy to use, and can really be leveraged in a meaningful way.

I'm always looking to pick up a new stack or a new technology, especially if I get to collaborate with others in the learning process. Whether I am in front of a screen or in a tent after a day in the mountains, you will usually find me thinking about how to make the latest project just a little bit better.

Projects

A few of my recent projects.

SimplyNote

A note-editing and organization application inspired by Evernote. Notes are taggable, searchable, and autosave when you are not looking.

SimplyNote is built with React/Redux, Ruby on Rails and PostgreSQL

Live Repo

Pathfinder.js

An interactive visualizer of four common pathfinding algorithms on 2D grid - A*, Greedy Best-First Search, Dijkstra and Breadth First Search.

Pathfinder.js is built with Vanilla JavaScript, CSS, and just a bit of jQuery.

Live Repo

Schemer

A visual database design tool that automatically generates SQL code. Create tables, columns and relationships by dragging and selecting elements in the editor.

Schemer is built with Vue.js, JointJS, Express/Node.js and MongoDB
